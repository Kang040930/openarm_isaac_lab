# 双臂 OpenArm 抓取实验设计

## 任务描述

双臂机器人根据指令用指定一只手抓取平台上的方块，移动到随机目标位置。

| hand_side=1 | 只用右手 |
| 方块初始 | (0.25, 0, 0.34) |
| 目标范围 | x=(0.15~0.3), y=(-0.1~0.1), z=(0.15~0.35) |

## 奖励函数 (对齐 OpenArm 单臂 demo)

**设计原则**：和单臂 lift 完全一致 — 逼近 + 二值举升 + 目标追踪 + 基础惩罚。

### 策略输出说明

策略输出 16 维动作 (左手7关节 + 右手7关节 + 左手夹爪 + 右手夹爪)，网络结构 [128,64,32] MLP。

### 逼近 (per-hand, 正向激励)

```
hand_side = 0 (左手模式):
  left_reaching  = 1.1 x (1 - tanh(左手距离/0.1))
  right_reaching = 0

hand_side = 1 (右手模式):
  left_reaching  = 0
  right_reaching = 1.1 x (1 - tanh(右手距离/0.1))
```

### 举升 (二值)

```
lifting = 15.0 IF z > 0.5 ELSE 0
```

### 目标追踪 (举升后生效)

```
goal_tracking      = 16.0 x (1 - tanh(距目标/0.3)) IF z > 0.5
goal_tracking_fine =  5.0 x (1 - tanh(距目标/0.05)) IF z > 0.5
```

### 惩罚

| 惩罚 | 权重 | 原因 |
|---|---|---|
| action_rate | **0** | IK 控制天然平滑 |
| left_joint_vel | -1e-6 | 极轻 |
| right_joint_vel | -1e-6 | 极轻 |

## 设计决策

| 决策 | 原因 |
|---|---|
| IK 控制 | 探索效率 100x，可部署到真实机器人 |
| 无 action_rate 惩罚 | IK 天然平滑，robosuite 不设此惩罚 |
| 无 object_dropping 终止 | 完整梯度信号，robosuite 风格 |
| 64 维视觉 (非 256) | 减少死神经元浪费，256->64 释放 75% 网络容量 |
| 双手相对位置观测 | 策略直接知道"手离方块多远"，不需从关节推算 |
| hand_side 遮挡奖励 | 策略学会按标签选动手 |

---

## 已知问题

| 问题 | 解决方向 |
|---|---|
| IK 稳定性 (NaN) | 减小步长 0.05->0.03 |

| 域态随机化缺失 | 未来加方块大小/质量/摩擦随机化 |
| 无视觉输入 | Phase 2 加入 Vision 编码器 |
